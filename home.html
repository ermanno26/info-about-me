Ciao, questo è il mio primo sito in HTML.

l'HTML, acronimo di "HyperText Markup Language", è utilizzato per la struttura logica di pagine web, e dal CSS, ovvero la rappresentazione grafica. É possibile programmare in HTML tramite un qualsiasi editor di testo, il blocco note ne è un esempio. Ovviamente se andiamo a scaricare editor di testo appositi, avremo vantaggi, quali la suddivisione dei tag in colori, visualizzare direttamente il risultato grafico tramite il software e moltre altre cose che variano di editor in editor.
